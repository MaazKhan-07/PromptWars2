📖 About the Project
ElectIQ is an educational web application built to demystify the election process for everyday citizens. Whether you're a first-time voter or just curious about how democracy works, ElectIQ walks you through every stage — from candidate filing to inauguration — with interactive tools, a smart AI assistant, a glossary, and a knowledge quiz.
This project was built as part of the Google Virtual Prompt Wars Challenge, using AI-assisted design and development workflows (Google Stitch + Antigravity) alongside the Anthropic Claude API for the intelligent chat assistant.

✨ Features:

🏠 Hero Section
Animated floating civic icons (🗳️ 📋 📅 ✅)
Scroll-triggered fade-in entrance animation
Direct CTAs to the learning flow and AI assistant

📅 Interactive Election Timeline
7-stage horizontal timeline (vertical on mobile)
Clickable cards that expand with details, key facts, and typical dates
Color-coded by phase: Registration → Primaries → Campaign → Voting Day → Counting → Certification → Inauguration

📚 Election Steps Explorer
11 detailed step cards with icons, descriptions, and inline accordions
Filter tabs: All | Federal | State | Local
Responsive CSS grid layout

🤖 AI Chat Assistant (Powered by Claude)
Real-time conversational assistant using the Anthropic Claude API
Non-partisan, fact-based responses about elections and civic processes
Pre-loaded question chips for instant interaction
Typing indicator, error handling, and smooth chat UX

📖 Glossary
20+ election terms defined in plain English
Live search filtering
A–Z alphabetical index navigation
💡 "Did You Know?" callout highlights

🧠 Quiz — Test Your Knowledge
8 multiple-choice questions across key election topics
Animated progress bar, instant feedback, and score summary
Emoji-rated results (🏆 / 🎓 / 📚) with motivational messages
Full reset and retake functionality

🌙 Additional UX
Dark mode toggle (persisted in localStorage)
Fully responsive: 375px / 768px / 1440px
Smooth scroll navigation
Scroll-triggered animations via Intersection Observer
Mobile hamburger menu

🛠️ Tech Stack
Structure: HTML
Styling: CSS (Custom Properties, Flexbox, Grid, Keyframe Animations)
Logic: JavaScript 
Fonts: Google Fonts — Inter + Sora

⚖️ Disclaimer
It is for education and learning purpose, and no real data used.
ElectIQ is a non-partisan, educational tool. All content is general in nature and intended for civic education purposes only. It does not represent any political party, candidate, or government body. Always verify election-specific details — including registration deadlines, polling locations, and voting rules — with your local or national election authority.
